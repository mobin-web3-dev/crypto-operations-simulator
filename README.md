# Crypto Operations Simulator

A simple web-based simulator designed to demonstrate common cryptocurrency operations workflows.

## Project Goals

* Simulate transaction monitoring
* Demonstrate deposit investigations
* Simulate withdrawal status
* Display blockchain confirmation progress
* Show common operational issues
* Practice crypto support scenarios

## Features

* Search Transaction ID (TXID)
* Transaction Details
* Confirmation Counter
* Network Detection
* Deposit Status
* Withdrawal Status
* Wallet Health
* Risk Indicator
* Operations Notes

## Technologies

* HTML
* CSS
* JavaScript
